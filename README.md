# LumiSign Connect

LumiSign Connect is an assistive AI system for real-time sign language recognition and communication support for deaf and hard-of-hearing users.

## Project Timeline

- Initial idea and research: January 2026
- Dataset collection: February 2026
- Model training: March 2026
- Web application development: April 2026
- GitHub upload and documentation: June 2026

## Project Description

The project was developed over several stages, starting with research and dataset preparation, followed by machine learning model training and interface development. The GitHub repository was created later to organize and publish the final implementation.

## Files

- `collect_web.py`: Collects sign language training data using the camera.
- `train_model.py`: Trains the machine learning model.
- `connect_app.py`: Runs the sign language recognition application.
- `lumisign_model.pkl`: Saved trained model.
- `sign_data.csv`: Dataset used for training.

## Technologies Used

- Python
- OpenCV
- MediaPipe
- Machine Learning
- Real-time Camera Processing
